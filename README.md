# todayinfo

**todayinfo** — это веб-сайт, созданный для того, чтобы предоставить вам всю ключевую информацию о сегодняшнем дне в одном месте. Проект объединяет данные о погоде, актуальные новости и текущие курсы валют, чтобы вы могли быстро сориентироваться в начале дня.

## О проекте

Цель проекта — предложить простой и удобный интерфейс для получения самой необходимой информации. Вместо того чтобы открывать несколько разных сайтов или приложений, вы можете зайти на todayinfo и узнать:

*   **Погоду:** Текущая температура, влажность, скорость ветра и прогноз на день.
*   **Новости:** Последние и самые важные новостные заголовки.
*   **Курсы валют:** Актуальные курсы основных мировых валют по отношению к вашей локальной валюте (например, USD, EUR к RUB).
